## 游戏简介  
你将扮演一个摄影师，受一个老人的委托，用一台相机为老人和他的老房子留下相片作为纪念。
而因为你的相机有着改变时间的能力，于是在完成老人委托的过程中，你逐渐了解他从幼稚到老年的一生回忆......

**开发引擎**：Unity 2022.3 
**美术风格**：低多边形3D (部分模型使用Unity资产商城的资产)
